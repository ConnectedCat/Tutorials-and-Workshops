Tracking images with Three JS and AR JS

This is a simple example of how to track images in AR using Three.js and AR.js. The code is based on the AR.js library.

Useful links:

  • NFT Marker Creator

Starting point

We'll start with the template described in Intro to Three JS. You can stop before Example 2, or complete the entire tutorial.

HTTPS Certificate and self-signed SSL certificate

To enable HTTPS on your local development server, you can use a self-signed SSL certificate. This is useful for testing AR applications that require a secure context. The process is outlined in Vite documentation.

npm install -D @vitejs/plugin-basic-ssl

Create (or update, if you already have one) your vite.config.js file. For more details see Vite documentation.
